A linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML, vistos até este momento:
Recrusos Tags associadas Descrição
Links <a> Criar links entre paginas
Listas <ul> e <li>
<ol> e <li>		 Criar listas de itens ordenados
Criar listas de itens nao ordenados
Imagens <img> Inserir imagens
